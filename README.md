# Easy-Coloc

## Project Overview

Easy-Coloc is a web application that helps roommates manage shared living expenses.  
Users can create or join a colocation, add shared expenses, and automatically calculate balances between members.

## Problem

Managing shared expenses in a colocation can be confusing.  
It is often difficult to know who paid, who owes money, and how much each person should contribute.  
Easy-Coloc centralizes expense tracking and provides clear balance calculations.

## Main Features

- User registration and login  
- Create and join a colocation  
- Manage members  
- Add and categorize expenses  
- Automatic balance calculation  
- Owner and member roles  

## Technologies

- Laravel  
- MySQL  
- Blade / Tailwind CSS
- Breeze for auth
